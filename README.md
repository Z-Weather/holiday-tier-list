# Holiday Tier List Platform

A platform for users to create, edit, and share personal holiday preference tier lists with drag-and-drop functionality and data analytics.

## Project Structure

```
holiday-tier-list/
├── backend/          # Node.js + Express API
├── frontend/         # React + Redux UI
├── docker/          # Docker configurations
├── docs/            # Documentation
└── README.md        # This file
```

## Tech Stack

- **Frontend**: React, Redux, Material-UI
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Cache**: Redis
- **Deployment**: Docker, Kubernetes
- **Cloud**: AWS

## Getting Started

1. Install dependencies in both backend and frontend directories
2. Set up MongoDB and Redis
3. Start the backend server
4. Start the frontend development server

## Features

- Create and edit holiday tier lists
- Drag-and-drop interface for ranking
- Share lists with other users
- Comment and interact with lists
- Data analytics and trends
- Responsive design
